# Trading Dashboard Assets Skill

이 스킬은 고성능 모듈형 자동매매 시스템의 웹 기반 대시보드 UI를 개발하거나 디자인을 구상할 때 활용할 수 있는 시각적 자산과 상세 설명을 제공합니다.

## 1. 주요 기능 및 활용 시점

*   **UI 디자인 참조**: 실제 거래소 수준의 전문적인 자동매매 대시보드 UI 디자인을 시각적으로 참고할 때.
*   **설계 지침**: 디자인에 반영된 핵심 기능(실시간 차트, 전략 설정, 종목별 모니터링 등)의 상세 설명을 이해할 때.
*   **개발 가이드**: 프론트엔드 개발 시 UI 컴포넌트 구성 및 레이아웃 구현에 대한 영감을 얻을 때.

## 2. 스킬 활용 가이드

### 2.1. 웹 대시보드 UI 디자인 이미지

`templates/auto_trading_web_ui_design.png` 파일을 통해 고해상도 웹 대시보드 UI 디자인 이미지를 확인할 수 있습니다. 이 이미지는 다크 모드 테마, TradingView 스타일 차트, 동적 전략 설정 패널, 종목별 스레드 모니터링 등 핵심 기능을 시각적으로 구현한 것입니다.

### 2.2. UI 디자인 상세 설명 문서

`references/web_ui_design_description.md` 파일을 참조하여 디자인 이미지에 포함된 각 UI 요소의 기능, 디자인 컨셉, 기술적 구현 제언에 대한 상세한 설명을 확인할 수 있습니다.

## 3. 스킬 사용 예시

사용자가 '자동매매 대시보드 디자인 보여줘', '트레이딩 UI 레퍼런스 필요해', '웹 기반 자동매매 화면 구성 어떻게 할까?'와 같은 요청을 할 때 이 스킬을 활성화하여 관련 디자인 자산과 설명을 제공할 수 있습니다.
