# Trading UI Technology Stack Guide

## 1. Frontend Frameworks
*   **React.js**: Recommended for complex state management and component reusability.
*   **Tailwind CSS**: Ideal for rapid UI development with a professional financial aesthetic.

## 2. Real-time Charting
*   **TradingView Lightweight Charts**: Industry standard for high-performance, interactive financial charts.
*   **Highcharts Stock**: Comprehensive library for advanced technical indicators.

## 3. Communication
*   **WebSockets (Socket.io)**: Essential for real-time bidirectional data streaming (Prices, Order Book, Logs).
*   **FastAPI (Python)**: High-performance backend to bridge Python trading engines with Web UIs.

## 4. State Management
*   **Zustand / Redux Toolkit**: To manage rapidly changing real-time data across multiple components.
